How does Eventloop work?

Event loop is responsible for executing the code, collecting and processing events and executing queued sub-tasks. Event loop affects both synchronous and asynchronous events.The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks.
